## ğŸ”‘ Key Principles

1. **One File, One Job**
   - GameScene.js = main container
   - PlayerController.js = just movement
   - NPCSystem.js = just NPC behavior
   - Easy to understand and debug

2. **Event-Driven Design**

   - React emits, Phaser listens
   - Complete decoupling
   - No hard dependencies
